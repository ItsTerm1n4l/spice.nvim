================================================================================

prime is a colorscheme for NeoVim written in Lua with built in support
for native LSP, TreeSitter and many more plugins.

                                        -LSP Trouble

                                        -Indent-Blankline.nvim
                                        -Vim-Which-Key

                                        -Dashboard

                                        -Neogit
                                        -Vim-Sneak
                                        -Nvim-Dap
                                        -Vim-Illuminate
                                        -lightspeed.nvim
                                        -mini.nvim

    + Ability to have a different background on sidebar windows like Nvim-Tree,
      Terminal, Packer, Whichkey, QuickFix etc.

    + Asynchronous highlight loading which makes the theme extremely fast

    + Added functions for live style switching, without the need to reload
      NeoVim

2. Usage                                                   *prime-usage*

    To use the default settings, just apply the theme

    If you are using Vim-Script:
        colorscheme prime~

    If you are using Lua:
        require('prime').set~


    prime also includes a Lualine theme. Set it in your Lualine config
        require('lualine').setup {~
          options = {~
            -- ... your lualine config
            theme = 'prime'~
            -- ... your lualine config
          }~
        }~
    Just make sure to set the theme to "prime", as "prime" already
    exist built in to lualine
    
    If the lualine's background has become transparent, make sure that
    lualine setup is being called after the prime theme has been applied.
